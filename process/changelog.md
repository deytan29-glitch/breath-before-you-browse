# changelog

- 2026-02-24: Project created — "Breath Before You Browse." A single-breath pacer that interrupts compulsive screen use. Built with vanilla Canvas + JS, no libraries, runs from file://. 5-state machine (IDLE → INHALE → HOLD → EXHALE → REFLECT → IDLE) driving one filled circle. Tap/click to start. Three selectable breathing levels (Calm 4-4-6, Balance 4-7-8, Deep 5-5-8). Phase-colored background with dual-hue system, gradient-filled circle, ambient halo, glowing progress ring, countdown timer, non-gamified breath counter.
